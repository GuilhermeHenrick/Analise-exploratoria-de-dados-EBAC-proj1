## 📌 Sobre o Projeto
Este projeto realiza uma **Análise Exploratória de Dados (EDA)** detalhada sobre problemas de logística urbana. O foco principal é entender a distribuição de entregas, identificar gargalos operacionais e otimizar a alocação de recursos entre diferentes hubs de distribuição.

## 🚀 Desafios Técnicos Superados
- **Data Wrangling de JSONs Aninhados:** Manipulação e normalização de dados complexos em formato JSON (flattening/unnesting) para transformá-los em DataFrames estruturados.
- **Tratamento de Geolocalização:** Limpeza e validação de coordenadas geográficas (latitude e longitude) para análise de rotas e distritos.
- **Limpeza de Dados:** Tratamento de valores ausentes, correção de tipos de dados e normalização de strings para garantir a integridade da análise.

## 📊 Insights e Visualizações
- **Distribuição de Entregas por Região:** Uso de gráficos de barras e contagem para identificar os hubs com maior volume de demanda.
- **Análise de Densidade:** Visualizações para entender a concentração de entregas e como a distância afeta a operação logística.
- **Correlações:** Exploração de variáveis que impactam o tempo ou a eficiência das entregas utilizando `Seaborn` e `Matplotlib`.

## 🛠️ Stack Tecnológica
- **Linguagem:** Python
- **Manipulação de Dados:** Pandas, Numpy
- **Visualização:** Seaborn, Matplotlib
- **Formato de Dados:** JSON / CSV

## 📂 Estrutura
- `analise_logistica.ipynb`: Notebook contendo todo o pipeline de extração, limpeza e visualização de dados.
